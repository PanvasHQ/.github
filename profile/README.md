# Panvas

Think. Sketch. Write. Build.

Panvas is a visual research workspace built for engineers, researchers and creators.

## Vision

Combine the best parts of:

- Notion
- Excalidraw
- Obsidian

into one visual-first workspace.

## Features

- Infinite Canvas
- Research Notes
- Workspace Organization
- Markdown
- Code Blocks
- LaTeX
- Local First
- Cloud Sync
